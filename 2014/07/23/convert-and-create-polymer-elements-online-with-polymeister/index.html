Convert and create Polymer elements online with Polymeister

Just a quick shoot-out to Polymeister, an online tool that allows you to generate Polymer elements (nested ones even!) easily. It also handles .slim and is currently under development by my good friend and partner in crime Karl Patrik Johansson.

Be sure to check it out once in a while to try out more features as the project progresses.
